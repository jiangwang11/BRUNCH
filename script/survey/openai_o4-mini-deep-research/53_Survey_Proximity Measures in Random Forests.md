# 引言  
随机森林（Random Forest, RF）通过计算训练样本在集成中共享叶节点的频率得到样本间的邻近度（proximity），这一度量被用于可视化、聚类、缺失值插补和异常检测等任务。然而，传统RF邻近度存在不准确反映模型学习到的数据几何结构的问题 ([arxiv.org](https://arxiv.org/abs/2201.12682#:~:text=forest%20proximities%20do%20not%20accurately,such%20as%20data%20imputation%20and))。近年来，多项研究提出新的邻近度定义或基于邻近度的算法，旨在提高近邻表示的质量与应用价值。本文综述2022–2025年间的代表性工作，分析其研究问题、核心方法和关键实验结论，阐述实验共性发现，并展望未来研究趋势与挑战。

# 方法分类与代表作  

- **几何保持型邻近度量与流形学习**：Rhodes 等 ([arxiv.org](https://arxiv.org/abs/2201.12682#:~:text=forest%20proximities%20do%20not%20accurately,such%20as%20data%20imputation%20and))针对传统RF邻近度不保留数据几何的问题，提出了**RF-GAP**定义，使基于邻近度的加权回归或投票精确匹配RF的OOB预测。他们证明RF-GAP严格保持了随机森林学习的数据结构几何，并在数据插补和异常检测任务中优于标准邻近度 ([arxiv.org](https://arxiv.org/abs/2201.12682#:~:text=proximities%20called%20Random%20Forest,with%20the%20learned%20data%20geometry))。基于此，Rhodes 及其团队将该理念扩展到监督流形学习：Rhodes (2023) ([arxiv.org](https://arxiv.org/abs/2307.01077#:~:text=these%20methods%20are%20limited%20to,based))指出无监督流形学习忽略标签信息会失去判别能力，提出用保留数据几何的RF邻近度作为监督流形嵌入的核初始化，实验证明此方法普遍提升了局部结构保留与嵌入质量 ([arxiv.org](https://arxiv.org/abs/2307.01077#:~:text=of%20class,based%20algorithms))。类似地，Rhodes 和 Rustad (2024) ([arxiv.org](https://arxiv.org/abs/2411.15179#:~:text=strengths,domain%20baselines.%20Experiments%20across%20multiple))使用RF几何邻近度构建对齐图，用于多领域数据的流形对齐，在跨域分类任务上显著优于无监督方法。这类方法的核心在于将RF邻近度的几何性质与标签信息结合，用于生成更可靠的低维嵌入。

- **基于邻近度的分类算法（尤其是时间序列）**：Lucas 等人提出了*Proximity Forest*（PF1.0），将RF思想应用于时间序列分类，通过对时间序列之间的距离度量进行随机化分裂，显著提升了大规模数据的分类效率和准确率 ([link.springer.com](https://link.springer.com/article/10.1007/s10618-024-01085-0#:~:text=overall%2C%20some%20specialized%20approaches%20are,ADTW%7D%5C%2C%7D%7D%5C%29%29%3B%20and%20%283%29%20cost)) ([research.monash.edu](https://research.monash.edu/en/publications/proximity-forest-an-effective-and-scalable-distance-based-classif#:~:text=milliseconds,allowing%20us%20to%20leverage%20the))。其后，Herrmann 等(2023) ([arxiv.org](https://arxiv.org/abs/2304.05800#:~:text=similarity,similarity%20measures%20employed%2C%20reducing%20the))提出**Proximity Forest 2.0 (PF2.0)**，集成了更高效的弹性相似性计算（如提前终止策略），并引入Amerced DTW相似度和代价函数调优等新技术，精简了基分类器的种类。实验表明，PF2.0在UCR时间序列基准上对比PF1.0显示出更高的准确率，并在多个特定数据集上超越了顶尖的核方法和神经网络 ([arxiv.org](https://arxiv.org/abs/2304.05800#:~:text=similarity,similarity%20measures%20employed%2C%20reducing%20the))。Shaw 等(2024) ([arxiv.org](https://arxiv.org/abs/2410.03098#:~:text=RF,outliers%2C%20comparing%20with%20nearest%20neighbor))则将RF-GAP思想扩展到近邻森林，并开发了**PF-GAP**算法。该方法通过多维尺度分析（MDS）对时间序列进行嵌入，并将PF-GAP邻近度与局部异常因子（LOF）结合用于异常检测。结果发现，基于PF-GAP的森林邻近度在链接误分类样本与异常点上比传统最近邻距离更强 ([arxiv.org](https://arxiv.org/abs/2410.03098#:~:text=those%20obtained%20using%20various%20time,outliers%20than%20nearest%20neighbor%20classifiers))，展示了基于改进的RF邻近度的分类方法在特定任务下的优势。

- **解释性与异常检测的应用**：近年来也有研究将RF邻近度用于模型解释和异常点分析。Rosaler 等(2023) ([arxiv.org](https://arxiv.org/abs/2310.12428#:~:text=We%20initiate%20a%20novel%20approach,prediction%20across%20observations%20in%20the))首次将RF预测权重形式化：他们证明RF预测可被表示为加权KNN，其中权重即样本间的RF邻近度。基于这一发现，作者提出了一种邻近度驱动的局部解释方法，可针对每个预测给出训练样本层面的归因，与传统特征归因方法（如SHAP）互补 ([arxiv.org](https://arxiv.org/abs/2310.12428#:~:text=We%20initiate%20a%20novel%20approach,prediction%20across%20observations%20in%20the))。此外在可视化方面，Mazumdar 等(2021) ([www.mdpi.com](https://www.mdpi.com/2079-9292/10/22/2862/htm#:~:text=%28MDS%29,However%2C%20we%20extend%20their)) ([www.mdpi.com](https://www.mdpi.com/2079-9292/10/22/2862/htm#:~:text=proximity%20value%20of%20an%20instance,effective%20solution%20for%20users%20to))将训练集样本通过基于RF邻近度的MDS映射到低维平面上，得到**RF相似度图谱**。他们发现，与其他样本邻近度低的实例往往恰好是被模型错误分类的“难例”，而高邻近度形成的“知识簇”对应模型已学到的高置信度关系，辅助解释了随机森林的全局和局部行为。这些工作表明，利用RF邻近度可以直观识别可疑样本和数据内在结构，为模型可解释性和异常点检测带来了新工具。

# 实验与评价总结  
综合各项实验结果，改良的RF邻近度量带来了几项共性收益：一方面，几何保留邻近度（如RF-GAP）使得基于邻近度的后续学习（如插补、嵌入、可视化）严密对应于原模型的输出，结果更加一致可靠 ([arxiv.org](https://arxiv.org/abs/2201.12682#:~:text=proximities%20called%20Random%20Forest,with%20the%20learned%20data%20geometry)) ([arxiv.org](https://arxiv.org/abs/2307.01077#:~:text=of%20class,based%20algorithms))；另一方面，在时间序列分类场景中，PF2.0等基于邻近度的新型分类方法在UCR基准上对比PF1.0和其它数据驱动方法表现出更低的误差率 ([arxiv.org](https://arxiv.org/abs/2304.05800#:~:text=similarity,similarity%20measures%20employed%2C%20reducing%20the)) ([link.springer.com](https://link.springer.com/article/10.1007/s10618-024-01085-0#:~:text=overall%2C%20some%20specialized%20approaches%20are,ADTW%7D%5C%2C%7D%7D%5C%29%29%3B%20and%20%283%29%20cost))。关于异常检测和解释性，多个研究一致观察到：基于RF邻近度的度量通常能将被误分类的样本与真实的异常样本联系起来，优于传统欧氏或DTW距离 ([arxiv.org](https://arxiv.org/abs/2410.03098#:~:text=those%20obtained%20using%20various%20time,outliers%20than%20nearest%20neighbor%20classifiers)) ([www.mdpi.com](https://www.mdpi.com/2079-9292/10/22/2862/htm#:~:text=proximity%20value%20of%20an%20instance,effective%20solution%20for%20users%20to))；利用邻近度的可视化也能区分“强知识簇”与难例，从而揭示模型的局部信息和置信度分布。本节中的引用 ([arxiv.org](https://arxiv.org/abs/2201.12682#:~:text=proximities%20called%20Random%20Forest,with%20the%20learned%20data%20geometry)) ([arxiv.org](https://arxiv.org/abs/2304.05800#:~:text=similarity,similarity%20measures%20employed%2C%20reducing%20the)) ([arxiv.org](https://arxiv.org/abs/2410.03098#:~:text=those%20obtained%20using%20various%20time,outliers%20than%20nearest%20neighbor%20classifiers)) ([www.mdpi.com](https://www.mdpi.com/2079-9292/10/22/2862/htm#:~:text=%28MDS%29,However%2C%20we%20extend%20their)) ([www.mdpi.com](https://www.mdpi.com/2079-9292/10/22/2862/htm#:~:text=proximity%20value%20of%20an%20instance,effective%20solution%20for%20users%20to)) ([arxiv.org](https://arxiv.org/abs/2310.12428#:~:text=We%20initiate%20a%20novel%20approach,prediction%20across%20observations%20in%20the))均佐证了上述结论。

# 趋势与挑战  
1. **解释性与信任延伸**：随着可解释AI的需求增强，基于邻近度的解释和信任分数预计将获得更多关注。未来研究可能将RF邻近度与特征归因方法整合，或用于评估模型不确定性，为决策提供更丰富的依据 ([arxiv.org](https://arxiv.org/abs/2310.12428#:~:text=We%20initiate%20a%20novel%20approach,prediction%20across%20observations%20in%20the))。  
2. **范式泛化与跨域应用**：已有工作将RF邻近度推广到时间序列（日益增多的**Proximity Forest**变体）和多域数据（流形对齐），未来可能进一步扩展到图数据、序列数据和元学习框架中，实现邻近度在各种监督度量学习场景下的通用性。特别，“广义近邻森林”等新模型将融合RF邻近度与其他模型的优势，为缺失值插补、异常检测等任务提供更强的跨领域能力 ([arxiv.org](https://arxiv.org/abs/2410.03098#:~:text=RF,outliers%2C%20comparing%20with%20nearest%20neighbor)) ([arxiv.org](https://arxiv.org/abs/2411.15179#:~:text=strengths,improved%20accuracy%20on%20downstream%20classification))。  
3. **效率与可扩展性优化**：尽管新型邻近度在精度上有优势，其计算代价是挑战。未来趋势可包括开发近似邻近度（如减少树深度或采样对）或硬件加速实现，使得在大规模数据集或实时应用中使用RF邻近度成为可能。  
4. **理论分析与深度融合**：理论上理解RF邻近度与数据几何、模型容量的关系仍是空白。未来研究可能证明新的邻近度性质（如RF-GAP的几何一致性）的一致性界限，并探索将邻近度思想与深度网络或混合模型结合以增强表达能力。  
5. **应用多样化**：邻近度概念有望进入更多应用场景，如半监督学习、集成学习解释、图神经网络邻接的相似度度量等，形成跨领域的研究热点。  

# 结论  
近年来的研究从多个角度提升了随机森林邻近度的表达能力与实用性。一方面，新定义的邻近度（如RF-GAP）严格保持了RF学到的几何结构，并在插补与可视化等任务上取得优势；另一方面，基于邻近度的分类算法（如PF2.0）在特定任务中也显现出超越传统方法的潜力。邻近度在模型解释、异常检测和低维嵌入中的创新应用进一步丰富了随机森林的分析工具。总的来看，2022–2025年的工作共同表明，通过改进邻近度的度量方法，RF在泛化性能与可解释性方面都获得了实质提升。未来研究可沿着加强可解释性、多模态应用和算法可扩展性等方向深入，推动邻近度在机器学习各领域的广泛应用。
