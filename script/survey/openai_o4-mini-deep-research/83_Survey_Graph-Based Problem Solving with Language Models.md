# 引言  
图结构数据广泛存在于现实世界（如社交网络、知识图谱等）中，对象间关系的推理往往可归结为图问题。近年来大型语言模型（LLM）在多项推理任务上取得突破，但其显式处理图结构问题的能力尚不明确 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=and%20perform%20structured%20operations%20remains,the%20benefit%20of)) ([research.google](https://research.google/pubs/talk-like-a-graph-encoding-graphs-for-large-language-models/#:~:text=%28LLMs%29%20remains%20an%20understudied%20problem,the%20correct%20choice%20of%20encoders))。例如，Wang 等人提出 NLGraph 基准，关注 LLM 能否从文本描述中**显式理解图结构**并执行诸如连通性、最短路、最大流等算法操作 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=and%20perform%20structured%20operations%20remains,the%20benefit%20of))。本文综述 2022–2025 年间利用语言模型解决图问题的代表性工作，按照方法类型分类讨论各类关键研究，并总结实验评价规律和未来趋势。

## 方法分类与代表作  
- **图问题基准与 LL***M 分析***：Wang 等人（2023）提出 **NLGraph** 基准，包含29,370个图问题，覆盖连通性、最短路、最大流、模拟图神经网络等8类任务 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=NLGraph%20,diminishes%20on%20more%20complex%20graph))。他们以 GPT-3/4 为代表，评测了多种提示策略，发现**LLM 对简单图任务已有初步能力**，但随着问题复杂度增加，高级提示（如链式思维）带来的提升显著下降，并易受图结构与问题设置中的偶然相关性干扰 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=models%20do%20demonstrate%20preliminary%20graph,to))。针对这一问题，作者提出了 *Build-a-Graph Prompting* 和 *算法提示* 等方案，将输入图显式化并指导算法思路，实验显示性能可提升3.07%–16.85% ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=spurious%20correlations%20in%20graph%20and,and%20evaluation%20code%20are%20available))，但最复杂的图任务依然无法有效解决。此类工作强调**评估 LLM 图推理能力**并设计专属提示策略的重要性。

- **图结构编码与提示策略**：Fatemi 等人（2024）研究如何将图结构编码为文本以供 LLM 推理（*TalkLikeAGraph*） ([research.google](https://research.google/pubs/talk-like-a-graph-encoding-graphs-for-large-language-models/#:~:text=%28LLMs%29%20remains%20an%20understudied%20problem,the%20correct%20choice%20of%20encoders))。他们对比了不同图到文本的编码方法（如边列表、邻接表等）、多种图推理任务和不同图结构，发现这三个因素**显著影响 LLM 表现** ([research.google](https://research.google/pubs/talk-like-a-graph-encoding-graphs-for-large-language-models/#:~:text=%28LLMs%29%20remains%20an%20understudied%20problem,the%20correct%20choice%20of%20encoders))。基于实验证据，该工作提出了针对性编码策略，如在文本提示中保留图的拓扑信息与权重，以提升推理准确率 ([research.google](https://research.google/pubs/talk-like-a-graph-encoding-graphs-for-large-language-models/#:~:text=%28LLMs%29%20remains%20an%20understudied%20problem,the%20correct%20choice%20of%20encoders))。此类研究表明，正确的图结构表示方式对 LLM 完成图问题至关重要。

- **基于图的推理框架**：针对 LLM 容易发生的“幻觉”问题，Jin 等人（2024）提出 **Graph-Chain-of-Thought (Graph-CoT)** 框架 ([aclanthology.org](https://aclanthology.org/2024.findings-acl.11/#:~:text=10%20domain%20graphs,We))。Graph-CoT 引入图执行作为中间步骤，与 LLM 交互迭代推理。每轮推理包含三步：LLM 进行文字推理、LLM 与显式图结构交互、最后在图上执行相关算法 ([aclanthology.org](https://aclanthology.org/2024.findings-acl.11/#:~:text=10%20domain%20graphs,We))。该模块化框架可插拔于任意 LLM 管道中，通过让模型不断访问和更新图信息来提高知识密集型任务的可靠性，并减少不根据信息的生成错误 ([aclanthology.org](https://aclanthology.org/2024.findings-acl.11/#:~:text=10%20domain%20graphs,We))。此类方法通过让 LLM 在“图上思考”来增强多步推理能力，预示着图结构推理与链式思维的有机结合值得进一步探索。

- **图问题的指令微调模型**：近期研究通过指令微调构造专用 LLM 来解决图问题。Chen 等人（2024）构建了 **GraphInstruct** 数据集，汇集了多种图问题及其显式推理路径，并以此训练了 **GraphWiz** 模型 ([www.emergentmind.com](https://www.emergentmind.com/papers/2402.16029#:~:text=to%20tackle%20a%20broad%20spectrum,and%20model%20performance%2C%20highlighting%20the))。GraphWiz 经指令调优后能够输出清晰的解题过程：在九个不同复杂度的图任务上，其带偏好对齐优化的版本（GraphWiz-DPO）平均准确率达到65%，远超 GPT-4 的43.8% ([www.emergentmind.com](https://www.emergentmind.com/papers/2402.16029#:~:text=resolving%20various%20graph%20problem%20types,and%20model%20performance%2C%20highlighting%20the))。这一结果表明，针对图问题的结构化指令数据显著提升了 LLM 的解题能力。类似地，Wu 等人（2024）提出了 **InstructGraph** 框架 ([aclanthology.org](https://aclanthology.org/2024.findings-acl.801#:~:text=Do%20current%20large%20language%20models,potential%20hallucination%20problems%20in%20graph))：他们首先将图数据转换为统一的类代码格式，然后对 LLM 进行图中心的指令调优并进行负样本偏好对齐训练 ([aclanthology.org](https://aclanthology.org/2024.findings-acl.801#:~:text=Do%20current%20large%20language%20models,potential%20hallucination%20problems%20in%20graph))。实验结果表明，InstructGraph 在多种图推理与生成任务上均获得**最优性能**，其平均表现比 GPT-4 提升了13%以上 ([aclanthology.org](https://aclanthology.org/2024.findings-acl.801#:~:text=generation%20tasks,respectively))。上述工作展示了通过大规模指令微调来“教会”LLM理解和操作图结构的可行性。

## 实验与评价总结  
从众多实验结果看，LLM 在简单图问题上通常能够给出正确答案，但随着图规模和推理深度增加，其准确率下降明显 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=models%20do%20demonstrate%20preliminary%20graph,to))。高级提示和链式思维能够在一定程度上帮助模型分解问题，但面对复杂的最大流、哈密顿回路等任务时，改进效果有限 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=models%20do%20demonstrate%20preliminary%20graph,to))。引入图结构的显式推理（如 Graph-CoT）或利用结构化指令微调能够显著提升性能：例如定制的 GraphWiz 模型和 InstructGraph 框架均相较于原始 LLM 提供了大幅度增益 ([www.emergentmind.com](https://www.emergentmind.com/papers/2402.16029#:~:text=resolving%20various%20graph%20problem%20types,and%20model%20performance%2C%20highlighting%20the)) ([aclanthology.org](https://aclanthology.org/2024.findings-acl.801#:~:text=generation%20tasks,respectively))。然而，这些模型仍易受到提示格式和数据分布的影响，错误关联和“幻觉”现象尚未被完全消除 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=models%20do%20demonstrate%20preliminary%20graph,to)) ([aclanthology.org](https://aclanthology.org/2024.findings-acl.801#:~:text=generation%20tasks,respectively))。总体而言，将图结构严格融入推理过程（无论是通过设计链式/图式思维过程，还是通过训练阶段给模型提供多样的图示例）是提升 LLM 图解题能力的关键。

## 趋势与挑战  
1. **图式思维的进一步发展**：未来可能涌现更多类似 Graph-CoT 甚至 Graph-of-Thought 的推理框架，通过在思考过程中允许模型并行探索多条推理路径（形成有向无环图）来提高复杂任务的求解效率。  
2. **专门化图任务语言模型**：随着 GraphWiz、InstructGraph 等工作的出现，为图问题“量身定制”的指令微调模型成为趋势。这类模型可能在任务特定的数据集上继续扩展，甚至催生类似GPT-4的“图版”大模型。  
3. **多模态与知识图谱融合**：为了弥补 LLM 仅靠文本推理的局限，研究将探索将图结构与视觉、表格等多种模态信息结合，或直接与大型知识图谱结合查询，以提供更精准的背景知识支持。  
4. **高质量基准与解释性研究**：新型大规模图推理基准（如丰富多变的现实网络场景）将不断出现，同时社区也会更加关注模型错误类型和推理过程的可解释性。相应地，可靠性保障（如对抗训练抵御“幻觉”）也是重要挑战。

## 结论  
综上所述，当前 LL★M 在图问题求解上展现了一定潜力，但性能仍受限于复杂度和结构化信息的呈现方式。已有研究表明，通过设计专门的提示与推理框架 ([paperswithcode.com](https://paperswithcode.com/paper/can-language-models-solve-graph-problems-in#:~:text=spurious%20correlations%20in%20graph%20and,and%20evaluation%20code%20are%20available)) ([www.emergentmind.com](https://www.emergentmind.com/papers/2402.16029#:~:text=resolving%20various%20graph%20problem%20types,and%20model%20performance%2C%20highlighting%20the))，或者对模型进行图任务定制的指令微调 ([www.emergentmind.com](https://www.emergentmind.com/papers/2402.16029#:~:text=resolving%20various%20graph%20problem%20types,and%20model%20performance%2C%20highlighting%20the)) ([aclanthology.org](https://aclanthology.org/2024.findings-acl.801#:~:text=generation%20tasks,respectively))，可以明显提升图推理效果。未来工作需进一步挖掘语言模型与图结构结合的方式，以解决现有方法容易出现的错误关联和幻觉问题，推动 LLM 在图算法、图搜索和知识图谱推理等领域取得更大突破。
