# 引言  
情感分析是自然语言理解的重要任务，用于识别文本的情绪倾向。传统方法依赖人工标注的情感词典，现代方法则以深度模型为主流。近年来，大规模预训练模型（如BERT）和卷积神经网络（CNN）在情感分类上取得显著进展，但二者各有所长。研究者尝试将情感词典中的先验知识融入深度模型，以弥补上下文模型对情绪词识别的不足。本文回顾了2022–2025年间结合情感词典的BERT和CNN情感分析模型的代表性工作，归纳其方法类别、核心技术及实验结论，并展望未来发展趋势。  

# 方法分类与代表作  

**1. 预训练语言模型（BERT等）+词典知识**。这类方法通过修改预训练或微调过程，将情感词典信息注入到模型中。Zhao等人提出*KESA*（2022）通过两种辅助任务将情感词与整体情感极性关联：一个任务根据句子情感选择正确的情感词（sentiment word cloze），另一个根据词性极性预测句子情感（conditional sentiment prediction）。该方法在多数据集上优于直接微调的基线，并可与现有知识增强技术叠加使用 ([arxiv.org](https://arxiv.org/abs/2202.12093#:~:text=Though%20some%20recent%20works%20focus,com%2Flshowway%2FKESA))。Milintsevich等（2024）关注心理健康领域，**在BERT模型输入中标记情感/情绪词**，用于抑郁症症状估计任务。他们发现，将情感、情绪及领域特定词典信息显式加入对话或社交媒体文本后，可以提升Transformer模型的预测性能，并在治疗师访谈的抑郁评估上达到了新的状态-of-the-art ([arxiv.org](https://arxiv.org/abs/2404.19359#:~:text=This%20paper%20explores%20the%20impact,therapist%20interviews))。Zhu和Mao（2025）提出**领域词汇知识增强的嵌入学习**方法，以解决小样本情感分类问题。他们发现BERT词嵌入对关键情感词的区分不足，于是将领域词典知识用于重投影模型嵌入，使同类文本表示更接近、异类更分散。实验表明，该知识增强的嵌入显著提高了情感与情绪分类准确度 ([arxiv.org](https://arxiv.org/abs/2506.01621#:~:text=Pre,specific%20lexical%20knowledge.%20The))。此外，Rizinski等（2023）研究金融领域的情感分析，提出**可解释词典（XLex）**框架。该方法结合Transformer模型和SHAP解释技术自动构建金融情感词典，生成的词典不仅扩展了传统Loughran-McDonald词典覆盖，并在金融数据集上优于原始词典，同时显著加快推理速度，增强了可解释性 ([arxiv.org](https://arxiv.org/abs/2306.03997#:~:text=novel%20methodology%20named%20eXplainable%20Lexicons,Thirdly%2C%20we%20illustrate%20that))。上述工作共通点在于，在Transformer架构内直接或间接地融合了情感词典信息，实验均显示加入外部知识可提升模型性能，同时提升结果的可解释性。  

**2. 卷积神经网络（及其他深度模型）+词典特征**。这类方法主要利用CNN、LSTM等网络结构结合情感词典特征。早期经典工作如Shin等（2016）在CNN上引入**词典嵌入和注意力机制**：对词向量和词典向量分别进行卷积，并通过注意力层对文档进行全局表示。结果表明，即使使用较小的词向量维度，融入词典信息的CNN模型在多项基准数据集上也能达到甚至超过当时最先进方法的性能 ([arxiv.org](https://arxiv.org/abs/1610.06272#:~:text=analysis%20that%20integrates%20lexicon%20embeddings,noisy%20words%20for%20sentiment%20analysis))。Lei等人（2018）提出**多情感资源增强注意力网络（MEAN）**，利用情感词典、否定词、情感强度词等多种资源，通过注意力机制融合到深度模型中。该网络将不同类型的情感知识看作独立通道，以丰富对否定和情感强度的表达，实验结果显示MEAN较传统基线具有稳健优势 ([arxiv.org](https://arxiv.org/abs/1807.04990#:~:text=Deep%20learning%20approaches%20for%20sentiment,robust%20superiority%20over%20strong%20competitors))。另一方面，也有以词典为基础的新型聚合方法：Lange等人（2022）提出**Lex2Sent**无监督框架，在没有标注数据的情况下对文本进行二分类情感分析。该方法使用词典词嵌入与文档嵌入之间的距离进行分类，通过重采样产生集成效应以提高鲁棒性。实验证明，Lex2Sent在二分类任务上性能优于传统词典方法，为稀数据或无标注场景提供了一种有效替代 ([arxiv.org](https://arxiv.org/abs/2209.13023#:~:text=BERT%20and%20zero,task%20of%20binary%20sentiment%20analysis))。这些方法均表明，在浅层网络或无监督模型中引入情感词典或多种情感资源，可以在一定程度上弥补上下文模型的语义盲点，提升分类效果。  

# 实验与评价总结  
各工作普遍在公开或行业数据集上验证其有效性，并与纯数据驱动模型进行对比。总体而言，引入词典知识的方法往往在准确率、F1值上取得**一致改进**：例如，XLex方法生成的词典在金融情感分类任务中超越了传统词典 ([arxiv.org](https://arxiv.org/abs/2306.03997#:~:text=novel%20methodology%20named%20eXplainable%20Lexicons,Thirdly%2C%20we%20illustrate%20that))；Lex2Sent模型在二分类任务上也超越了单纯的词条计数方法 ([arxiv.org](https://arxiv.org/abs/2209.13023#:~:text=BERT%20and%20zero,task%20of%20binary%20sentiment%20analysis))。一些研究还强调了效率和可解释性的提升：XLex构建的词典使最终模型显著更小更快，且基于规则的词典决策路径提高了分析结果的透明度 ([arxiv.org](https://arxiv.org/abs/2306.03997#:~:text=novel%20methodology%20named%20eXplainable%20Lexicons,Thirdly%2C%20we%20illustrate%20that))。然而，性能提升的幅度依赖于任务和数据集：在大规模标注数据充足的情况下，深度模型本身已很强大，词典增强贡献較小；而在领域词汇稀缺或长距离语境复杂的任务中（如金融文本、医疗对话），结合词典往往带来更明显的增益。如Zhu等指出，当关键词在情感判断中起关键作用时，BERT原生嵌入的区分度不足，此时通过领域词典加投影的方式能够有效提升分类性能 ([arxiv.org](https://arxiv.org/abs/2506.01621#:~:text=Pre,specific%20lexical%20knowledge.%20The))。此外，Milintsevich等还观察到，不同词典（情感词典、情绪词典、领域词典）对不同任务影响不同，表明词典选择和集成策略也是影响性能的重要因素 ([arxiv.org](https://arxiv.org/abs/2404.19359#:~:text=This%20paper%20explores%20the%20impact,therapist%20interviews))。总的来说，实验一致表明**融合情感词典知识可改进情感模型表现**，但其效果受数据规模、领域特征和词典质量等因素的制约。  

# 趋势与挑战  
展望未来，我们预测情感分析领域将出现以下趋势。首先，**解释性和可解释性将成为热点**。XLex等工作表明，将可解释性融入词典构建可以提升用户信任度 ([arxiv.org](https://arxiv.org/abs/2306.03997#:~:text=novel%20methodology%20named%20eXplainable%20Lexicons,Thirdly%2C%20we%20illustrate%20that))；更多研究可能借鉴模糊逻辑或专家系统（如2025年预测的词典-模糊混合框架 ([arxiv.org](https://arxiv.org/abs/2510.15843#:~:text=language,The%20framework))），实现既精准又可解释的情感打分。其次，**少样本与跨域适应**需求增强。Zhu等（2025）通过领域词典改进嵌入，显示在小样本场景下词典补充很有效 ([arxiv.org](https://arxiv.org/abs/2506.01621#:~:text=Pre,specific%20lexical%20knowledge.%20The))；未来研究可进一步探索自动构建领域词典的技术，如利用大模型进行词典扩展，或结合对比学习提升领域泛化能力。第三，**大模型与词典协同**将更广泛地被研究。当前工作（如KESA）通过辅助任务将情感词知识注入预训练 ([arxiv.org](https://arxiv.org/abs/2202.12093#:~:text=Though%20some%20recent%20works%20focus,of%20label%20combination%20methods%20are))；未来可能出现更多轻量级插件或适配器，使巨型语言模型在情感分析中利用外部情绪知识，兼顾强泛化与少量域知识。第四，**多语言与多模态融合**趋势明显。跨语种情感分析需要构建跨语言词典并在多语言BERT上应用，已有SemEval挑战表明改进多语预训练策略的重要性 ([arxiv.org](https://arxiv.org/abs/2306.01093#:~:text=2023,2023))。此外，多模态情感分析可利用视觉情绪词典和情感标志（如表情、语调）等辅助信息，这一方向也在兴起。最后，**情感细粒度与强度分析**受到关注。未来研究可能结合情感强度词典和上下文模型，精细化地预测情绪极性和强度，甚至深入讽刺或幽默情感识别等复杂任务。  

# 结论  
近年来，将情感词典知识与BERT/CNN结合的研究不断涌现。代表性工作通过在预训练或网络结构中注入词典信息，实现在多种情感分析任务上提升性能和可解释性的目的。总体看，词典增强在长尾领域、少样本和需要解释的场景中尤为有效 ([arxiv.org](https://arxiv.org/abs/2306.03997#:~:text=novel%20methodology%20named%20eXplainable%20Lexicons,Thirdly%2C%20we%20illustrate%20that)) ([arxiv.org](https://arxiv.org/abs/2506.01621#:~:text=Pre,specific%20lexical%20knowledge.%20The))。尽管大模型已取得巨大成功，但情感分析仍受限于领域词汇和上下文变化，未来研究仍需在模型能力和知识注入之间寻找平衡。我们预计未来将有更多研究关注**动态、可解释的词典构建**、**跨域知识迁移**和**与大语言模型的深度融合**等方向，以推动情感分析技术的发展。
