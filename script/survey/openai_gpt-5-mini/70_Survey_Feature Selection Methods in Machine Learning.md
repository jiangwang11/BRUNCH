引言  
特征选择（feature selection）仍是面向高维、稀疏与小样本数据时保证模型可解释性与泛化性的核心步骤。近三年（2022–2025）工作呈现两个明显方向：一是基于信息论/过滤器准则的精细化度量与复合准则（例如联合互信息与联合熵的结合）；二是基于集成树/启发式搜索与深度网络/超网络的端到端嵌入式方法以应对非线性与小样本问题。本文以严格可检索的文献为依据，对 2022–2025 年间代表性方法做分类综述，比较其方法学差异与实验结论，并提出未来若干可验证的研究趋势与挑战。所有被选代表作均为公开发表论文或可检索技术报告；每篇介绍限定 4–6 句（突出问题、核心方法、关键实验结论）。

方法分类与代表作  
（每类 3–5 篇代表工作，按发表年份/主题组织）

A. 信息论与联合互信息类（Filter / 互信息改进）  
1) 基于最大化联合互信息与最小化联合熵：刘亚文、温勇 (2023) — “JMIMJE”[hanspub.org].  
- 研究问题：在互信息筛选中，单一互信息或联合互信息在特征子集数量变多时难以甄别冗余且可能选择稳定性差的子集。  
- 核心方法：提出 JMIMJE，两阶段先用联合互信息（JMI）筛出候选，再以联合熵（joint entropy）作为稳定性指标做二次筛选，最终以 M/H（互信息/熵）准则选择特征。  
- 关键实验结论：在多种 UCI 数据集上的贝叶斯分类器实验显示，JMIMJE 在小样本或低维候选数时比 mRMR、JMI 更稳、平均预测精度略有提升；论文给出联合互信息与联合熵随子集大小变化的实证分析以说明动机。[pdf.hanspub.org](https://pdf.hanspub.org/AAM20230400000_26612659.pdf)

2) 信息论改进与标准化互信息：郭艾堃 (2022) — “基于高维复杂数据的变量选择方法研究”[hanspub.org].  
- 研究问题：传统线性累加互信息方法在高维复杂数据上会高估冗余并忽略类别变量的信息对称性。  
- 核心方法：提出基于标准化互信息（symmetric uncertainty / NMI）与极值（min / max）准则替代简单累加的过滤式算法（NMI-JMI 风格），并利用交互信息来调节冗余度估计。  
- 关键实验结论：在 11 个公开数据集（含生物与图像数据）上，用 KNN/SVM/RF 比较显示该方法在多数场景下能提升宏平均 F1，与若干信息论基线相比在显著性测试上占优。[hanspub.org](https://www.hanspub.org/journal/paperinformation?paperid=51887)

3) 基于统计推断的变量选择：丁宁 (2025) — “高维数据下基于统计推断的变量选择方法研究”[hanspub.org].  
- 研究问题：面向超高维数据，如何将统计推断的理论性（收敛性、Oracle 性质）与计算可行性结合以选择变量。  
- 核心方法：提出一种以稀疏约束与自适应迭代为核心的变量选择框架，强调先筛选相关变量再采用特定降维/矩阵优化以降低估计矩阵复杂度。  
- 关键实验结论：在结构化与非结构化高维模拟与真实数据上，方法据称改善了偏差并提供稳健估计；论文包含收敛性与计算复杂度理论分析。[pdf.hanspub.org](https://pdf.hanspub.org/sa2025143_282581577.pdf)

B. 集成树 / 启发式混合策略（Tree + metaheuristic / wrapper）  
4) X-ACO：结合 XGBoost 与蚁群优化的特征选择（张凌翱，2023）[hanspub.org].  
- 研究问题：在基于树的特征重要度存在相关性偏倚时，如何通过启发式搜索找到更小且低冗余的子集。  
- 核心方法：用 XGBoost 的特征重要度作为蚁群算法的启发信息，并用特征间皮尔森相关系数作为路径“长度”调节信息素更新，蚁群搜索得最优子集（X-ACO）。  
- 关键实验结论：在多个 UCI 数据集上，以逻辑回归为分类器比较，X-ACO 在减少特征数的同时保持或略增准确率，且在 AUC/F1 等指标上通常优于 Relief 与原始全特征基线；作者报告对相关特征的抑制效果。[image.hanspub.org](https://image.hanspub.org/Html/22-1542860_64728.htm)

5) 基于最大联合互信息最大化（JMIM/JMIM 变体）与启发式权重调整（相关文献综述）  
- 研究问题与方法学脉络：近年来多个工作尝试将联合互信息的“整体相关性”与动态互信息权重相结合（例如 JMIM、JMIM 的改进），以降低盲目累加冗余的副作用。代表性实现多在 2015–2023 年间提出并被后续 JMIMJE、NMI-JMI 等工作引用；相关效果常以小样本数据集和多分类文本/生物数据验证。[见上文与引用文献]

C. 小样本 / 深度嵌入式与超网络方法（Embedded / End-to-end）  
6) 超网络（Hypernetwork）设计用于高维小样本特征选择：Wei/董/余 等（SRnet，2025）[rhhz.net].  
- 研究问题：小样本高维场景下，直接训练大参数网络会严重过拟合，如何用超网络端到端产生主网络权重以减少首层参数并实现稳健特征选择。  
- 核心方法：提出稀疏重构网络（SRnet），由分解层（奇异值嵌入）、重构层（稀疏重构得分）与关联层（稀有增强）组成的三个辅助网络并行生成主网络首层权重，采用端到端交叉熵+重构+稀有度正则共同训练。  
- 关键实验结论：在 12 个生物医学小样本高维数据集上（样本数 50–203，特征数 >2000），SRnet 在平均分类准确率上较 7 种对比模型（含 WPFS、FsNet、dietnet 等）提高约 2–3 个百分点，并通过消融实验证明分解层贡献最大；论文提供详细消融与超参数分析。[html.rhhz.net](https://html.rhhz.net/tis/html/202402018.htm)

7) 工程化、轻量化的深度嵌入式方法（参考/对照）  
- 研究问题与方法学脉络：Concrete Autoencoders（可微分特征选择，自编码器框架，ICML 2019）与 LassoNet（JMLR 2021）等工作提出了可微分的嵌入式特征选择路径；SRnet 与 WPFS 在 2022–2025 年的工作中直接引用并扩展了“辅助网络生成主网络权重”的思路以适配小样本高维生物数据。SRnet 的实验也把这些方法作为基线进行对比（见上文）。（原始论文参考见参考文献部分）

D. 特征选择稳定性与集成（Stability / Ensemble methods）  
8) 稳定性研究综述与提升策略：李云（2012）—“稳定的特征选择研究”[chinaaet.com].  
- 研究问题：特征选择结果的可重复性/稳定性在可解释性场景（如基因选择）中至关重要。  
- 核心方法论结论：稳定性受数据扰动、算法机制、冗余特征与高维小样本影响；主要改进方向包括集成特征选择（ensemble）、样本加权与特征组群（consensus group）。  
- 关键实验结论：综述并归纳了稳定性度量指标与两个提升稳定性的实用方法类别，后续若干实证论文在构建稳健选择器时均采用这些策略作为组件。[chinaaet.com](https://www.chinaaet.com/article/200287)

实验与评价总结（共性结论，非逐篇复述）  
- 评价协议与可比性：多数论文在比较时采用 UCI / 生物医学 / 图像分类数据集、交叉验证或多次重复实验，并以分类准确率、宏/微 F1、AUC 与特征子集大小作主要衡量；少数论文还报告特征稳定性（子集重叠）与计算成本（FLOPs 或运行时间）。  
- 冗余控制 vs 子集稳定性的权衡：信息论类改进（如 JMIMJE、NMI-JMI）普遍尝试在“相关性提升”与“冗余抑制/稳定性提升”之间建立二元度量（互信息与熵或极值替代累加），实验显示在小样本与少量所选特征场景下能获得更稳定的分类性能（波动更小）。  
- 嵌入式深度/超网络方法对小样本有效但依赖工程细节：SRnet 等超网络通过大幅削减首层自由参数与引入重构/稀有度正则，在高维小样本生物数据上显著降低过拟合并提升平均准确率；但其超参数敏感性、训练稳定性与可解释性（如何映射到原始变量）仍需更多标准化评测。  
- 启发式+树结构方法在实践可用且鲁棒：X-ACO 等将 XGBoost 的重要度与元启发式搜索结合，可显著降低选特征数而保持/改善分类指标，适合工程化部署；但其全局最优性无法保证且随机性与种子敏感。  
- 评价盲区：跨数据集的统一稳定性度量、对“选择到的特征是否具备领域可解释性” 的人工验证、以及在不平衡/重标签任务下的方法健壮性评估仍较少见，导致方法迁移性评价不足。

趋势与挑战（2025 年前后可验证的预测，至少三点）  
1) 多准则信息论筛选将向“联合熵/条件熵 + 极值/对称化互信息”方向标准化与理论化。理由：JMIMJE、NMI-JMI 等实证表明仅用互信息或简单冗余累加已不能同时保证稳定性与相关性，未来会出现更通用的范式并配套渐近理论与离散/连续变量一致化处理方法（可验证：新范式在多数据集上显著降低子集熵与提高稳定性）。参考实例：[pdf.hanspub.org](https://pdf.hanspub.org/AAM20230400000_26612659.pdf)、[hanspub.org](https://www.hanspub.org/journal/paperinformation?paperid=51887)。

2) 超网络/辅助网络端到端选择将在小样本生物医学场景成为主流实践路线，但对“可解释性映射”提出更高要求。理由：SRnet（2025）展示了超网络能显著减少过拟合并提升准确率，但领域受众（生物/医学）要求将促使后续工作在权重→原始特征映射、可信度打分与可视化方面形成标准化工具链（可验证：后续工作公布可重复代码与领域专家评估结果）。参见：[html.rhhz.net](https://html.rhhz.net/tis/html/202402018.htm)

3) 组合策略（ensemble-filter + embedded fine-tune）及自动化搜索（AutoML for FS）将被更多采用以提升稳定性与迁移性。理由：单一策略难以兼顾稳定性、非线性建模与计算可行性；实践中常见的方案是先用稳定过滤器降维，再以超网络或树集成做精调，并用启发式搜索（如 X-ACO）在有限预算内优化最终子集（可验证：对比 pipeline 与单一方法在多领域任务的迁移误差）。参见：[image.hanspub.org](https://image.hanspub.org/Html/22-1542860_64728.htm)、[pdf.hanspub.org](https://pdf.hanspub.org/AAM20230400000_26612659.pdf)。

4) 标准化评测套件与稳定性基准将被提出并广泛采纳。理由：当前文献多用不同数据集与度量，不利于横向比较；社区需要包含“子集稳定性、可解释性评分、计算资源与迁移误差”的统一基准（可验证：若出现社区基准并被 10+ 篇论文采用，则预测成立）。

结论  
2022–2025 年间特征选择研究在方法学上呈现“双轨并进”——信息论/过滤器方向朝着更细致的联合度量（互信息＋熵/标准化）发展，能够在少量特征情形下提升稳定性；同时，工程化的嵌入式/超网络与启发式+树集成策略在非线性、小样本及工程化场景展现出实际价值。未来工作应优先解决（1）稳定性与可解释性的统一度量、（2）超网络生成权重到原始特征映射的可验证解释机制、（3）跨任务/跨域的比较基准化问题。本文所列代表作与结论可为后续方法设计与系统评测提供直接起点。
