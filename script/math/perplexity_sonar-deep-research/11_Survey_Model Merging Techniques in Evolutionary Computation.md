# Model Merging Techniques in Evolutionary Computation: A Comprehensive Survey (2022–2025)

本综述系统回顾了2022年至2025年间Model Merging与进化计算相结合的核心技术进展。基于权重空间参数化、进化算法驱动的自动化搜索、模式连通性几何约束以及多模态任务场景中的应用，我们识别出五大技术范式。关键发现表明：（1）进化驱动的模型合并能以50倍计算成本缩减实现高效的大语言模型融合，打破人工直觉的局限；（2）参数作用域对齐与符号冲突解决构成合并性能的关键瓶颈；（3）模式连通性的几何认知从线性插值扩展到非线性曲面连接，提升了理论基础；（4）标度律首次被量化，揭示专家数量对性能收益的幂律关系；（5）参数高效微调下的模型合并成为新的研究前沿。基于技术成熟度与应用潜力，我们预测未来研究将聚焦于：多模态基础模型的动态路由合并、跨域知识迁移的理论保证、以及与联邦学习的深度融合。

## 一、引言

Model Merging 作为一种后训练范式，无需额外标注数据与训练计算即可整合多个专家模型的能力，已成为基础模型开发的民主化工具。截至2025年1月，Hugging Face开放模型库中超30%的排名靠前模型采用了合并技术构建。然而，传统人工设计的合并配置（如权重插值系数、层级融合策略）严重依赖领域知识与反复试验，难以发现非直觉的最优组合。进化计算与Model Merging的结合代表了一种根本性范式转变：通过自然选择机制自动化探索参数空间和数据流空间中的组合策略，在消费级GPU上实现了原本计算上不可行的多目标优化。本综述基于2022年以来发表在顶级会议与权威期刊（ICLR、NeurIPS、ACL、CVPR、Nature Machine Intelligence）及arXiv平台的52篇代表性文献，系统梳理该领域的方法学演进、实验规律与开放问题。
